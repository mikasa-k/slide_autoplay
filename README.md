# スライドランチャー
発表会で発表者が交代する時間を最小限にするための、スライド自動再生ツールです。
パワーポイントとLibreOfficeのスライドとmp4動画に対応していますが、パワーポイントの普及率が高いのでwindowsで実行することが前提になります。

## 使い方
playlist.csv に発表順の情報を記入する。

スライドや動画を使わない場合は「file_name」のところに「実演」などと記載する使い方もできる。
```
file_name,speaker_name
```

## GUIの仕様
main.py を実行するとフルスクリーンで立ち上がります。(プレゼンテーションモード)

Ctrl+Qで画面を終了できます。
Enterキーまたはボタンで開始します。
